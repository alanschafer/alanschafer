About This Site

This is more than just a GitHub Pages site. It's a creative ground zero for my work — a place where fiction, identity, and memory all blur into something raw and personal.

I’m Älan Schäfer. I don’t come here to explain everything — I come here to build something that resonates, quietly but deeply. The work speaks for itself, but this page exists to give a bit of grounding.

The site launched with The 2ND Artifact, a surreal and fractured tale about trauma and what remains when memory fails. That’s the first piece — there are more artifacts coming.

I chose GitHub as a medium not just because it's simple and fast, but because it's oddly fitting — a repository for things pulled apart and pieced together again. Code and narrative aren’t so different.

This site will evolve. You’re welcome to watch it grow, share it, or reach out. I’m not building a brand. I’m building a signal.
